# 구현 로드맵

## Phase 1. 화면 구조 안정화

- 전체 피드 메신저형 UI 정리
- 하단 입력창 UX 확정
- 댓글/첨부 팝업 확정
- 참여자 패널 확정
- 모바일 메뉴 드로어 확정

## Phase 2. Supabase 연결

- Supabase 클라이언트 설정
- 게시판 목록 조회
- 피드 게시글 조회
- 게시글 작성
- 댓글 작성
- 첨부파일 Storage 연동

## Phase 3. 인증/권한

- Supabase Auth 연결
- 사용자 프로필 생성
- 팀별 게시판 접근 권한
- 대표/팀장 전체 열람 정책

## Phase 4. 업무게시판

- 기간 선택
- 목표/진행업무/이슈/다음 계획 입력
- 업무 기록 목록
- 전주 대비 Diff 표시 설계

## Phase 5. 개인 메모

- 개인 메모 작성
- 체크리스트
- 중요 표시
- 게시글 전환

## Phase 6. 알림

- 멘션 알림
- 댓글 알림
- 게시글 업데이트 알림
- 읽음/미읽음

1. 현재 mock data를 Supabase query 함수로 교체
2. 게시글 작성 폼 상태 연결

4. 첨부 버튼을 Supabase Storage 업로드로 연결
5. 모바일 화면별 실제 사용성 점검
