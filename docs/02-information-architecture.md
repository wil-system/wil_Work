# 정보 구조

## 전체 IA

```text
메인 협업 화면
├─ 전체 피드
├─ 팀별 게시판
│  ├─ 컨텐츠팀
│  ├─ 영업팀
│  ├─ 생산팀
│  ├─ 물류팀
│  ├─ 신사업팀
│  ├─ 경영관리실
│  └─ 대표이사
├─ 업무게시판
├─ 개인 메모장
├─ 알림센터
└─ 마이페이지
```

## 현재 화면 구성

### PC

- 좌측: 게시판 목록
- 중앙: 메신저형 전체 피드
- 우측: 토글식 참여자 패널
- 댓글/첨부: 중앙 피드 위에 팝업으로 표시

### 모바일

- 상단: 메뉴 버튼
- 좌측 드로어: PC 좌측 게시판 목록과 동일한 구조
- 중앙: 메신저형 피드
- 참여자: 우측 슬라이드 패널
- 댓글/첨부: 모바일에서는 게시글 아래 인라인 표시

## 사용자 흐름

### 전체 피드 작성

1. 사용자가 하단 입력창에 내용을 작성한다.
2. 필요 시 파일을 첨부한다.
3. `@담당자`, `#해시태그`를 포함해 등록한다.
4. 피드 하단 최신 대화로 표시된다.

### 댓글 확인

1. 게시글 하단 `댓글 n개`를 클릭한다.
2. PC에서는 중앙 팝업이 열린다.
3. 팝업 왼쪽에는 원문, 오른쪽에는 댓글이 표시된다.
4. 댓글 입력창은 댓글 영역 하단에 고정된다.
5. 답글 버튼을 누르면 해당 댓글 아래에 답글 입력창이 표시된다.

### 첨부 확인

1. 게시글 하단 `첨부 n개`를 클릭한다.
2. 첨부파일 전용 팝업이 표시된다.
3. 댓글 팝업과 첨부 팝업은 분리된다.

### 참여자 확인

1. 상단 `참여자` 버튼을 클릭한다.
2. 우측 패널이 열린다.
3. 참여자는 소속별로 그룹화되어 표시된다.
