# 제품 개요

## 목적

WIL 협업툴은 현재 카카오톡 단체방 중심으로 흩어지는 업무 커뮤니케이션을 하나의 기록 가능한 협업 공간으로 모으기 위한 내부 업무 도구다.

핵심 목표는 빠른 공유는 메신저처럼 가볍게 유지하되, 댓글, 첨부, 담당자 멘션, 팀별 게시판, 업무 히스토리를 통해 업무 맥락이 사라지지 않게 만드는 것이다.

## 해결하려는 문제

- 카카오톡 단톡방이 여러 개로 분산되어 히스토리 관리가 어렵다.
- 업무 이슈가 대화 속에 묻혀 담당자, 상태, 첨부파일을 다시 찾기 어렵다.
- 주간업무나 팀별 진행 상황을 대표/팀장이 한눈에 보기 어렵다.
- 개인 메모와 팀 협업 게시글 사이의 전환 흐름이 없다.

## 핵심 사용자

- 대표이사: 전사 이슈와 주간업무를 한눈에 확인
- 팀장: 팀별 업무 공유, 담당자 확인, 댓글 피드백
- 팀원: 전체 피드 공유, 댓글 대응, 업무 기록, 개인 메모

## 1차 MVP 범위

- 전체 피드
- 팀별 게시판
- 업무게시판
- 개인 메모장
- 댓글/대댓글
- 첨부파일
- 참여자 패널
- 알림센터 기본 구조

## 제외 또는 후순위

- 카카오톡 연동
- 실시간 알림
- 권한별 세부 정책 UI
- 파일 미리보기
- 주간업무 Diff 자동 분석
- 실제 검색 인덱싱
